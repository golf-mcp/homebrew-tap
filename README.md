# Golf MCP Homebrew Tap

Homebrew formulae for [Golf](https://golf.dev) tools.

## Installation

```bash
brew tap golf-mcp/tap
```

## Available Packages

| Formula       | Description                                              | Install                            |
|--------------|----------------------------------------------------------|------------------------------------|
| golf-local   | MCP security proxy for local AI tool connections         | `brew install golf-mcp/tap/golf-local`   |
| golf-scanner | Security scanner for MCP servers - discover and audit across IDEs | `brew install golf-mcp/tap/golf-scanner` |

## Usage

```bash
# Install golf-local
brew install golf-mcp/tap/golf-local

# Install golf-scanner
brew install golf-mcp/tap/golf-scanner

# Update
brew upgrade golf-local
brew upgrade golf-scanner

# Uninstall
brew uninstall golf-local
brew uninstall golf-scanner
```
