# Golf MCP Homebrew Tap

Homebrew formulae for [Golf](https://golf.dev) tools.

## Installation

```bash
brew tap golf-mcp/tap
```

## Available Packages

| Formula     | Description                                      | Install                          |
|------------|---------------------------------------------------|----------------------------------|
| golf-local | MCP security proxy for local AI tool connections | `brew install golf-mcp/tap/golf-local` |

## Usage

```bash
# Install golf-local
brew install golf-mcp/tap/golf-local

# Update
brew upgrade golf-local

# Uninstall
brew uninstall golf-local
```
